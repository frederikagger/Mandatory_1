CSS

CSS is the technology that makes a website come to life. CSS stands for "cascading stylesheet" which means that it is CSS that handles the layout of a page. CSS uses selectors (html classes, id's, elements) to define a set of rules for each. The layout that is ultimately shown on a webpage is determined by the specicity of the rules. The more specific the rules are the higher priority will it also have. E.g. will a class selector have higher priority than a element selector because the class is more specific.

jQuery

jQuery is widely use to make a website more interactive, but its popularity is slowly declining. That is because CSS have greatly improve over the last 10 years, so today it's possible to make animations and effects purely in CSS. Furthermore, the DOM API have greatly improved and the new fetch API have simplified API calls a lot, which means that there is almost no need for jquery anymore.

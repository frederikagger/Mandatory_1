SCSS

SCSS is a CSS preprocessor that makes it easier to write maintainable CSS code. One of the advantages is that SCSS makes it possible to seperate your code into different files and then import them into one main file instead of having it all in one long CSS file there could be thousands of lines of code. SCSS is compiled to CSS so it's just a way to better organize your CSS code. SCSS also make it possible to use variables which makes it easy to make different themes in your application. Another feature of SCSS is that you can nest your selectors inside eachother which makes it a lot easier to read and maintain.

JavaScript basics

Callbacks

A callback is a function that is being passed as a parameter into another function which is then executed inside that function. Callbacks are often used in asynchronous programming.

Functional vs OOP

There are a lot of functional methods in modern JavaScript that have advandages over OOP methods. E.g. map, filter, reduce instead of foreach, for loops and while loops. For a method to be considered functional it cannot have any sideeffects, and it has to return something (meaning it cannot return void).

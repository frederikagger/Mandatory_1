Backend basics

node.js

node.js is a JavaScript runtime that makes JavaScript run away from the browser.

npm

npm stands for "node packet manager" and is responsible for pulling the right dependencies into your project. To use npm in a node project you first have to create a file called package.json where npm will read dependencies from. To initalize a package.json you can either create it on your own, or use the npm CLI command "npm init" with an optional flag -y (-y for yes) to generate a package.json with default values. Then you have to run "npm i" where "i" stands for install, which you can also type out. "npm i" will make npm pull in the right dependencies into your project. If you want to add a package, (e.g. express.js) you can type the command "npm i express". That will pull in the latest version of express for you.

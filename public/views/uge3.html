API basics

http methods

http is a protocol that defines communication between a webserver and a client. The http protocol defines the following request as allowed requests.

  • GET
  • POST
  • PUT
  • DELETE
  • PATCH
    • It is good practice to use the http request as they are defined in the http protocol. A delete request should delete something on the server, a get request should return something from the server and so on.
